O portfólio demonstra não apenas habilidades técnicas, mas também capacidade de criar soluções escaláveis e seguir boas práticas de desenvolvimento.

## 🚀 O que você encontra no site

### 🏠 **Hero Section**
Apresentação direta como **Backend Java Developer** com foco em APIs REST e arquitetura de microsserviços.

### 👨‍💻 **Sobre Mim**
Trajetória desde plugins Minecraft (Java + Bukkit/Spigot) até desenvolvimento de APIs REST enterprise com Spring Boot. Conexão clara entre experiência com POO, eventos e persistência de dados.

### 💼 **Projetos Backend**

- **LS Optimizer** - Aplicativo desktop C# para otimização do Windows

- **Conceitos:** Clean Code, SOLID, Design Patterns
